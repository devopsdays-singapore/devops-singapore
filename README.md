### Join the Community

The **most effective way** to ask questions, suggest topics, propose ideas, or give feedback is to **open a GitHub Issue**.

- **Propose a talk** → [New CFP Issue](https://github.com/devopsdays-singapore/devops-singapore/issues/new?template=cfp.md)
- **Questions or suggestions** → Comment on an event issue or open a new one
- **Help organizing or volunteering** → Open an issue with label `help-wanted`
- **Sponsorship or jobs** → Open an issue with label `sponsor` or `jobs`

All upcoming events are listed on our **[Luma Calendar](https://lu.ma/devops-singapore)**.

We will announce our Discord server after a few successful events.

### How We Run Events
- One GitHub Issue = one event (kept open during planning and the event)
- Past events become closed issues (easy to search history)
- RSVPs are handled on **Luma**
- All planning and discussion happens transparently in the event issues

### How to Propose a Talk
We welcome talks about real-world experiences, tools, failures, and lessons learned.

1. Open a new issue using the [CFP template](https://github.com/devopsdays-singapore/devops-singapore/issues/new?template=cfp.md)
2. Or simply comment on an upcoming event issue with your idea
